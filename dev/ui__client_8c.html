Array  args,
Error *  error

Handler for "redraw" events sent by the NVIM server

This function will be called by handle_request (in msgpack_rpc/channel.c) The individual ui_events sent by the server are individually handled by their respective handlers defined in ui_events_client.generated.h

Note
The "flush" event is called only once and only after handling all the other events
Parameters
channel_idThe id of the rpc channel
uidataThe dense array containing the ui_events sent by the server
[out]errError details, if any
